# sistema de estudiante
-------------------------------------
Este proyecto  tiene como objetivo administrar una lista de estudiantes con los datos que el usuario proporciona.
La información que se recopila y se organiza para mostrar un resultadosal usuario como la lista de estudiantes y el promedio general del grupo.
A través de un menú principal, el usuario puede agregar, eliminar, actualizar y consultar los datos.
# Conjunto De Dato Personales
------------------------------------------
Este proyecto tiene como objetivo analizar información de los circuitos de Fórmula 1 utilizando los datos del archivo circuits.csv.
El archivo contiene columnas como latitud, longitud, altitud, país y nombre del circuito, entre otras.
El programa carga y organiza los datos para mostrar resultados como:
* La latitud máxima y mínima.
* Los nombres y países de los circuitos.
* La cantidad de circuitos por país.
* Una clasificación por altitud según la altura del circuito.
  Se utiliza la librería pandas para realizar todo el análisis y mostrar la información de manera clara y estructurada.
