# sistema de estudiante
-------------------------------------
Este proyecto está hecho en Python y tiene como objetivo administrar una lista de estudiantes con los datos que el usuario proporciona.
La información se recopila y organiza para mostrar resultados como la lista de estudiantes y el promedio general del grupo.
A través de un menú principal, el usuario puede agregar, eliminar, actualizar y consultar datos, lo que facilita la gestión
